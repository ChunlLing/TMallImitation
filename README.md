## 布局页面
* 首页（index.html）
* 分类页（classification.html）
* 产品页（details.html）
* 购物车页面（shoopingCart.html）
* 注册页面（register.html）
* 登录页面（login.html）

### 首页
实现的特效
* 导航栏下拉菜单鼠标经过的显示和隐藏
* 鼠标经过商品图的放大提示
* 商品图的轮播效果
* 滚动条scrollTop大于浏览器高度时顶部显示搜索框、左侧显示隐藏的导航条  

存在的问题
* 商品列表显示行不正确
